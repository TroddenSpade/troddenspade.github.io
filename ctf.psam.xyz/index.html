WTF is CTF ?

CTF (Capture The Flag) is a competition in which contestants have to solve a variety of tasks ranging from basic to expert programming exercises, such as file scavenging or decrypting a message. In these challenges, the contestant is usually asked to find a specific string that may be hidden on the server or behind a webpage. This goal is called the flag, hence the name!

How does it Work ?

Our Version of CTF is more based on solving intuitive challenges rather than skill-based challenges, so BauCTF is a team competition in which you have to get the highest score from all the challenges, the score is decided by the number of challenges done and by the time each challenge has taken.

Who is it for ?

As mentioned, BauCTF is more of an intuitive based challenge, so all you need to participate in our event is a basic programming knowledge, good intuition and out of the box thinking. if you think you have these qualities then stay tuned for our events, because you belong there.
